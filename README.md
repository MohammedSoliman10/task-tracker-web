# Task Tracker Web App

A web-based task tracker built with Python and Flask.

## Features
- Add, update, and delete tasks
- Mark tasks as in-progress or done
- Filter tasks by status
- Data stored in a JSON file
- Clean and responsive UI

## How to run

### 1. Install Flask
pip install flask

### 2. Run the app
python app.py

### 3. Open in browser
http://localhost:5000

## Tech Stack
- Python
- Flask
- HTML/CSS
- JavaScript
