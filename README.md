# Task Tracker Web App

A full-stack web application built with Python and Flask to manage daily tasks — now with user accounts, so everyone has their own private task list.

## Features
- User registration and login (with hashed passwords)
- Each user only sees their own tasks
- Add, update, and delete tasks
- Mark tasks as in-progress or done
- Filter tasks by status
- Data stored in a SQLite database
- Simple admin panel to view registered users
- Clean, modern, responsive UI

## Tech Stack
- Python
- Flask
- Flask-Login
- SQLite
- HTML / CSS / JavaScript

## How to run locally

### 1. Install dependencies
pip install -r requirements.txt

### 2. Set a secret key (optional but recommended)
set SECRET_KEY=your_random_secret_key

### 3. Run the app
python app.py

### 4. Open in browser
http://localhost:5000
